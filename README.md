# Laboratório de Estudos: Python e Back-end 🐍

Este repositório contém exercícios, rascunhos e projetos iniciais desenvolvidos durante a minha graduação em **Análise e Desenvolvimento de Sistemas (UNISUAM)** e cursos complementares.

## 🚀 Objetivo
O foco principal deste espaço é a fixação de sintaxe, lógica de programação e o domínio das ferramentas essenciais para um desenvolvedor Back-end.

## 🛠️ Tecnologias e Ferramentas
- **Linguagem principal:** Python 3
- **Versionamento:** Git & GitHub
- **Sistema Operacional:** Linux (Ubuntu)

## 📂 O que estou praticando aqui?
- Fundamentos de lógica de programação.
- Estruturas de dados e repetição.
- Manipulação de arquivos e diretórios.
- Fluxo de trabalho com Git (commits, branches e push).
